## Run Locally

Clone the project

```bash
  git clone https://github.com/satriadhikara/smartmedisync
```

Go to the project directory

```bash
  cd smartmedisync
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
